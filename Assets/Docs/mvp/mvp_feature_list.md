# MVP Feature List (Lean but Powerful)

## Goal

Validate that players enjoy movement + generative building + alignment
feedback loop.

------------------------------------------------------------------------

## 1. Core Movement (Primary Hook)

Movement must feel good within 30 seconds.

Minimum set: - fast sprint or glide - vertical mobility (jump boost /
air dash / grapple) - 1 "wow" mechanic (portal jump OR launch pad OR
midair dash reset) - physics that allow chaining movement

Success metric: players move for fun even without objectives.

------------------------------------------------------------------------

## 2. Small Procedural World

Keep scope controlled.

Requirements: - 1 biome only - procedural terrain (rolling hills or
floating islands) - points of interest scattered sparsely - basic
resource nodes

Avoid: - multiple biomes - complex climate systems - large map size

Players should experience: "what's over there?"

------------------------------------------------------------------------

## 3. Wavefunction Collapse Building System (Simplified)

Focus on output beauty, not system complexity.

Minimal version: - 6--12 modular building pieces - automatic snapping -
structure evolves visually as player adds blocks - buildings always look
coherent

Example modules: - wall - tower - arch - roof - pillar - connector piece

Critical: players should create something visually satisfying within 2
minutes.

------------------------------------------------------------------------

## 4. Alignment System (Binary Only)

Track:

good ←→ evil

Alignment changes based on: - crafting specific structures OR - placing
certain block types

Visual feedback: good structures → lighter shapes, open geometry evil
structures → dense, spiky, compressed geometry

------------------------------------------------------------------------

## 5. Structure Unlock Loop

Building should unlock new possibilities.

Example progression: structure size increases → unlock new piece

OR

good alignment unlocks: - glide boost structure - light tower

evil alignment unlocks: - mob spawning structure - underground expansion
piece

Limit to: 3 unlocks total.

------------------------------------------------------------------------

## 6. Minimal Mob Interaction

Include: 2 mob types max - 1 attracted to good alignment - 1 attracted
to evil alignment

Basic behaviors: - wander - approach structure - simple interaction
(follow, attack, orbit)

------------------------------------------------------------------------

## 7. Resource Loop

Keep simple.

Include: - 1 primary resource - 1 rare resource

loop: explore → collect → build → unlock

Avoid: - inventory complexity - crafting recipe overload

------------------------------------------------------------------------

## 8. Visual Identity (Fast to produce)

Retro low-poly style.

Keep: - simple color palette - simple lighting - strong silhouettes

Consistency \> detail.

------------------------------------------------------------------------

# Core Gameplay Loop (MVP)

move fast → discover area → collect resource → place structure →
structure evolves visually → alignment shifts → unlock new structure
piece → mobs react → explore more

------------------------------------------------------------------------

# Hard Scope Limits

DO NOT include in MVP: - multiplayer - combat depth - skill trees -
quests - narrative - large crafting trees - survival meters - multiple
biomes - weather systems

------------------------------------------------------------------------

# Suggested 4-week MVP roadmap

Week 1: movement prototype

Week 2: basic procedural terrain

Week 3: WFC building prototype

Week 4: alignment visual change + simple mobs

------------------------------------------------------------------------

# MVP Success Criteria

Strong signal: players experiment with structures voluntarily

Very strong signal: players show creations to others

Weak signal: players only follow objectives

Failure signal: players stop after 10 minutes
